## 🎯 ¿Cómo usar este repositorio?

Cada sesión contiene:
- 📖 **Teoría** — conceptos explicados con ejemplos reales
- 💻 **Código** — ejemplos listos para ejecutar
- 🧠 **Mini-test** — preguntas para verificar tu comprensión
- ✅ **Checklist** — lo que debes poder hacer al terminar

## 🤖 Aprende con IA

Este repositorio fue creado con ayuda de **Claude** (Anthropic). Puedes usar los archivos `.md` de cada sesión para aprender de forma interactiva:

1. Abre [claude.ai](https://claude.ai)
2. Comparte el contenido de la sesión que quieras estudiar
3. Dile a Claude que te evalúe con el mini-test al final de cada sesión
4. Claude te dará feedback detallado de tus respuestas

**Ejemplo de prompt para empezar:**
```
Hola Claude, voy a estudiar la Sesión 1 del curso de Java Concurrencia.
Aquí está el contenido: [pega el contenido del .md]
Por favor guíame por la sesión y al final evalúa mis respuestas del mini-test.
```
